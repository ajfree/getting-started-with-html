Learn how the Web Works

When working with the internet, there are things behind the scene that put a web page together.

HyperText Markup Language, commonly referred to as HTML, is the standard markup language used to create web pages.
It is written in the form of HTML elements consisting of tags enclosed in angle brackets.

Tags & Elements

Tags mark the beginning and end of an element.
Elements are the bits that make up web pages and refer to everything within a set of opening and closing tags.

Blocks & Incline

Block elements are placed to start a new line. Blocks may contain other blocks, as well as inline elements and data (content).
Inline elements are used to display on the same line and may contain only data and other inline elements.
